cnchar.js
功能全面、多端支持的汉字繁体字拼音笔画js库
View project on github
功能
拼音|音调
多音字|词
笔画数|笔序
繁体字|火星文
安装使用
import cnchar from 'cnchar';
'汉字'.spell();
'汉字'.stroke();
<script src="https://fastly.jsdelivr.net/gh/theajack/cnchar/dist/cnchar.latest.min.js"></script>

    '汉字'.spell();

</script>
